# PoC: Vue 2 & 3 Component Library with `vue-demi`

## Install

```sh
yarn install
```

## Vue 3 (works)

```sh
yarn dev:vue3
```

## Vue 2 (does not work)

```sh
yarn dev:vue2
```

### Error Message

```plain text
Uncaught SyntaxError: The requested module '/node_modules/.vite/vue.js?v=c2bff7da' does not provide an export named 'createElementBlock'
```
